## CV entry

**RentChain — Full-stack & Blockchain Portfolio Project**

Developed a privacy-first rental platform concept using Next.js, TypeScript, Solidity, Hardhat and ethers.js. Implemented SHA-256 booking proofs, smart-contract lifecycle management, wallet integration, optional deposit escrow, automated contract tests, CI and security documentation. Designed the architecture to keep personal booking data off-chain while enabling independent integrity verification.

**Technologies:** TypeScript, React, Next.js, Solidity, ethers.js, Hardhat, SHA-256, Docker, GitHub Actions.

## Interview pitch

“I built RentChain to explore where blockchain actually adds value in a normal business system. Instead of putting customer data on-chain, I keep operational data off-chain and register only a deterministic SHA-256 booking proof. The smart contract acts as an independent trust layer and explicit rental state machine. I also added optional deposit escrow, role restrictions, tests, CI and a small threat model. The main design decision was to use blockchain selectively rather than treating it as a replacement database.”
